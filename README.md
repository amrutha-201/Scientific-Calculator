# Scientific Calculator

A Scientific Calculator built using HTML, CSS, and JavaScript. It provides both basic and advanced mathematical operations through a simple and interactive user interface.

## Features

* Basic arithmetic operations: Addition, Subtraction, Multiplication, Division, and Modulus
* Trigonometric functions: Sin, Cos, Tan
* DEG and RAD mode support
* Logarithmic functions: log and ln
* Exponential calculations
* Factorial calculation
* Square root operation
* Power calculation (xʸ)
* Mathematical constants: π and e
* Answer recall (Ans)
* Backspace functionality
* Clear screen (AC)

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Start performing calculations.
